# GlyphJS Obsidian Plugin

BRAT distribution repository for the [GlyphJS](https://github.com/VledicFranco/glyphjs) Obsidian plugin.

## Install via BRAT

1. Install [BRAT](https://github.com/TfTHacker/obsidian42-brat) from Obsidian Community Plugins
2. Run **BRAT: Add a beta plugin** and enter:
   ```
   https://github.com/VledicFranco/glyphjs-obsidian
   ```
3. Enable **GlyphJS** in Settings → Community plugins
